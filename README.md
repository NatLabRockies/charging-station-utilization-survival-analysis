# charging-station-utilization-survival-analysis
Code and sample data for external sharing for the EV charging station utilization survival analysis.

The `.py` files showcase the code used to clean and combine the datasets use in the analysis, with confidential data redacted from the code.

The `data` directory includes sample L2 and DC data, with confidential columns not included.

The file titled `survival_analysis_time_of_day_and_day_type.R` is a file that showcases the code used to conduct the day type and time of day survival analysis included in the paper. This file can be run and will create and save plots based on the provided sample data.
